Detailed Description

Classe héritant de Tache permettant de manipuler des tâches unitaires.

Parameters
progression>0 et <100 : permettant de suivre la progression des taches preemtives après leur programmation, puisqu'on peut programmer une tache preemtive à plusieurs reprises
precedenceest un vector contenant toutes les taches qui précèdent la tache
suivanteest vector contenant toutes les taches qui viennent après la tache
